# Distributed-Computing-Spark-on-Kiva-Loan-Prediction
- Predicted how much time (in hours) a given loan will take to reach full funding in Python with a rudimentary Random Forest model using PySpark and Spark MLlib
- Deployed a pipeline to bridge Amazon S3, MongoDB server, and EC2 cluster for NoSQL data throughput
- Wrote a blog about How to Run an Application on Spark Standalone Cluster on Medium
